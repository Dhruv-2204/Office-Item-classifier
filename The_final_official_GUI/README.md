# Office Item Classifier GUI

For this project pyside6 was used due to its performance level which allows efficient rendering of video frames and overlays. It is easy to create complex interfaces with detection logs, confidence scores, and control panels. It works consistently on Windows, Linux, and macOS

Follow these instructions:

use interpreter pycharm 3.12 as it consists of all necessary packages to be download and it is compatible with most of them

# Installing necessary environment for the program
python -m venv venv
venv/Scripts/activate   #for windows

# Please be patient with the installation of libraries in requirements.txt file, it might take some time

pip install -r requirements.txt

run main.py  #starting the GUI program
